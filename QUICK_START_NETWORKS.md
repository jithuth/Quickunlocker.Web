## ?? **Add More Networks:**

### **Via Admin Panel (Easiest):**
1. Go to `/AdminPanel/Networks`
2. Click "Add New Network"
3. Fill form:
   - Select Country
   - Enter Network Name
   - (Optional) Logo URL
   - Display Order
   - Enable/Disable
4. Click "Save"

### **Via SQL:**
```sql
-- Example: Add MTN Egypt
DECLARE @EgyptId UNIQUEIDENTIFIER = (SELECT Id FROM Countries WHERE Code = 'EG');

INSERT INTO Networks (Id, Name, LogoUrl, CountryId, IsActive, DisplayOrder, CreatedAt, IsDeleted)
VALUES (NEWID(), 'MTN Egypt', NULL, @EgyptId, 1, 5, GETDATE(), 0);
```
